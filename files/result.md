# Summary of Financial Invoice Form PDF

The PDF is a **Thai Financial Invoice/Delivery Note Form** (ใบส่งของ/ใบแจ้งหนี้). It's a blank template form with the following structure:

## Document Fields

| Field | Thai | English |
|-------|------|---------|
| เล่มที่ | Volume/Book No. | ......... |
| เลขที่ | Document No. | ......... |
| วันที่/เดือน/พ.ศ. | Date/Month/Year | ......... |
| ผู้ขาย | Seller | ......... |
| ที่อยู่ | Address | ......... |
| นามลูกค้า | Customer Name | ......... |

## Invoice Table Columns

1. **ลำดับที่** - Item Number
2. **รายการ** - Description/Items
3. **จำนวน** - Quantity
4. **หน่วย** - Unit
5. **ราคา/หน่วย** - Price per Unit
6. **รวมเงิน** - Total Amount

## Footer Section

- **รวมเป็นเงิน** - Grand Total
- **จำนวนเงิน (ตัวอักษร)** - Amount in Words
- **ลงชื่อผู้รับของ** - Receiver's Signature
- **ลงชื่อผู้ส่งของ** - Sender's Signature
